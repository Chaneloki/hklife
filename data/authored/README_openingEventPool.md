# P1 W1-W6 opening event bible v5.1

- Event choices are authored per event + variant.
- No global personality choice bank should be merged into events.
- Render choice.text as a short button; playerLine is the fuller action/dialogue.
- statusDelta is integer core-state direction, not guaranteed reward. Most choices total 0–2, with more 0 and -1 paths for preserving, missing, resting, avoiding, or paying costs.
- skillExpDelta is skill experience, not skill level.
- dialogueTriggerPreview is generated from openingDialogue.
